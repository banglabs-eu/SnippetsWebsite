{S} Snippets

A knowledge base built around tags and sources.

Capture notes from anything you read, watch, or attend. Tag them. Cite them. Browse them through whichever lens fits the question.

01 / Concept

Browse your knowledge as a shape

Tag clouds

Click tags to filter. The cloud rebuilds in real time and fades non-matching tags, so co-occurring topics surface as you narrow down.

Cross-dimension filters

Switch between tag, source, author, and publisher views. Selections in one dimension carry over to filter what you see in the next.

Source-grounded notes

Every note can attach to a source with a locator — page, timestamp, URL fragment. Citations are generated from author and publisher data.

Markdown with tags

Notes are Markdown. Tags are per-user and free-form; each note can carry any number, edited inline without leaving the page.

02 / Features

Everything in one place

Tag autocomplete

Type to filter existing tags or create new ones. Enter and Tab add a tag; Backspace removes the last.

Typed sources

Books, lectures, videos, articles, podcasts, websites, papers. Each type shows the fields that matter — year and edition for books, date and location for lectures.

Ordered authors

Multiple authors per source, in citation order. Search by first or last name. Each author links back to every source they appear on.

Inline editing

Click a note's body, tags, or source to edit in place. Sources have an editable detail panel that exposes only the fields relevant to their type.

Anki export

Send the currently visible notes to Anki as an .apkg deck. Sources become the card front; note bodies become the back; tags travel with each card.

Sanitized Markdown

Bold, italic, lists, links, code blocks — rendered via marked and sanitized through DOMPurify so pasted HTML can't bite.

Dark and light themes

Toggle in the header. Persists across sessions; the URL ?theme= override is handy when you want to ignore your stored preference.

8 languages in the app

English, German, French, Spanish, Italian, Dutch, Polish, Danish. Auto-detected from your browser; switchable in the header.

Magic-link sign-in

No password required. Request a one-time link from your email to sign in. Google OAuth and password sign-in also supported.
